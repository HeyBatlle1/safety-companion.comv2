# **App Name**: Construction Guardian

## Core Features:

- User Authentication & Profiles: Multi-role authentication (admin, project_manager, field_worker) with user profile management and company assignment.
- Safety Checklists System: Dynamic, pre-shift safety checklists with scoring and historical data visualization.
- Project Management: Project assignment system and project-specific safety requirements.
- Safety Documentation: Safety report creation, submission, and safety violation tracking, enhanced by AI, where the AI tool determines which regulation or piece of information needs to be present in the report
- Certification Tracking: User certification management and expiration date tracking with alerts.
- Dashboard Analytics: Interactive dashboard with real-time safety status indicators.

## Style Guidelines:

- Primary color: Royal blue to highlight important actions.
- Background color: Metallic Grey for a modern, professional look.
- Accent color: Chrome and Glass to complement the primary blue, providing visual interest and contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look.
- Use clear, simple icons to represent different safety categories and actions.
- Use a grid-based layout with clear sections for each module (checklists, reports, analytics).
- Smooth transitions for page navigation and loading animations for data display. Animate the safety score gauges.
