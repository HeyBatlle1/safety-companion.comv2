# Construction Guardian

Construction Guardian is a proactive safety management platform designed for the modern construction industry. This application leverages AI-powered analysis to move beyond simple compliance checklists, providing predictive safety intelligence to prevent accidents, reduce risks, and protect your team on every project.

## Key Features

- **Dynamic Safety Checklists**: Create and manage pre-shift safety checklists with real-time scoring, critical item identification, and mandatory photo verification for high-risk tasks.
- **Intelligent Job Hazard Safety Analysis (JHSA)**: An AI-driven system based on OSHA requirements that analyzes site conditions, regulatory context, and visual data to identify hazards and recommend corrective actions.
- **AI-Powered Predictive Analytics**: Utilizes Gemini to analyze checklist and JHSA submissions, identifying hidden risk correlations, flagging anomalies, and providing actionable recommendations to prevent future incidents.
- **Comprehensive Reporting**: Automatically generate detailed reports for various stakeholders, including executive summaries, technical breakdowns for safety officers, and trend analyses for insurance purposes.
- **Centralized Dashboard**: Visualize key safety metrics, track open incidents, monitor checklist completion rates, and get a real-time snapshot of your team's safety performance.
- **Certification & Profile Management**: Keep track of team member certifications and get alerts for upcoming expirations.

## Tech Stack

- **Framework**: Next.js (App Router) & React
- **AI**: Google's Gemini via Genkit
- **UI**: ShadCN UI Components, Tailwind CSS, Lucide Icons
- **Styling**: Tailwind CSS with CSS Variables for theming
- **Deployment**: Firebase App Hosting
